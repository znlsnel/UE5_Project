# Description
- **프로젝트 소개** <br>
  언리얼 엔진을 활용하여 게임 개발 매커니즘을 학습하는 것을 목표로 시작한 프로젝트입니다.<br>
  밤이되면 나타나는 몬스터를 처리하는 디펜스 장르의 게임이며,<br>
  게임 머니를 통해 아이템을 구매하고 스킬을 업그레이드하며 끝까지 살아남는 게임입니다.

- **개발 기간** : 2023.02.27 - 2023.07.27
- **사용 기술** <br>
-언어 : C++<br>
-엔진 : Unreal Engine <br>
-개발 환경 : Windows 11<br>
  <br>
